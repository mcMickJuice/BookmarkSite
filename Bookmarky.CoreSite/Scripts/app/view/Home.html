Bookmarky

This is the bookmarky site. It is a pet project by Mike Joyce to consolidate bookmarks, saved Reddit posts, github repos and any other reference sources on the internet.

The motivation for this project is:

  • Build a SPA app from the ground up using Web API and Angular
  • Integrate modern webdev tools and preprocessor languages (Grunt, npm, SASS, CoffeeScript) into my workflow using Visual Studio
  • Create a site that consolidates, organizes and provides easy access to things I save on the internet

I hope to learn a lot from creating this site while also developing a useful tool

Key Features

This site will offer the following features:

  • Consolidate and Organize Bookmarks and Saved links
  • Adding of new bookmarks
  • Import of bookmarks from major browsers
  • Ranking and Reviewing of links
  • Syncing of bookmarks and links from various sources (Chrome, Reddit etc.)

To faciliate syncing of various sources, a service project will be implemented to check all configured sources
